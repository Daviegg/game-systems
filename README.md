## 🧩 Projeto principal

### Minigame de Moedas

Sistema de minigame onde o jogador entra em uma arena por tempo limitado e coleta moedas.

**Funcionalidades:**
- Sistema de entrada no minigame
- Timer de rodada (30 segundos)
- Spawn de moedas
- Sistema de pontuação
- Conversão de moedas em tickets
- Loja com itens
- Ranking de jogadores
